[Aprender habilidades]:

Felicidades! Finalmente has alcanzado el nivel 5.
Ahora puedes aprender una nueva habilidad. Para aprender una habilidad, debes encontrarte con el maestro en la ciudad. Sigue la flecha sobre tu cabeza para encontrarlo.
Para aprender una habilidad, se requiere una cierta cantidad de puntos de habilidad (SP). Tu SP aumentara cada vez que mates a un monstruo, asi que si alguna vez te quedas sin SP, puedes ganar mas cazando monstruos.
Para ver tu nivel de SP actual, abre la ventana de informacion del personaje. Si haces clic en el icono que parece una cara o presionas las teclas ALT+T, se abrira la ventana de informacion del personaje.
